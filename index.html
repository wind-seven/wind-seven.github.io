fractal block world

some call it the best worst game of all time

it's a simple RPG where the main gameplay loop is enter dungeon, defeat monster, get stronger, progress. what sets it apart is the mechanic of being able to shrink or grow using ring devices placed in the world, which makes for quite an interesting gimmick that would drive a lot of people in if not for the Questionable artstyle

general tips for playing the game:

  • when the game tells you (or when you discover that) you can find X upgrade in Y location, write it down, it'll save you numerous headaches
  • the above applies also for finding X location in Y location
  • don't worry too much about naming waypoints since the game automatically attaches to them a descriptor of where they are (i name all of mine "cum")
  • don't be afraid of respawning, while it does take away all of your ammo it leaves your gold so you can usually just find a buystation and restock, but do still load whenever possible

things you can do with a ti-84+ce

if you like to program but your school doesn't allow people to bring their laptops, you should get a ti-84+ce python edition. if you have one, you can jailbreak it (to unlock some features that Texas Instruments decided to kill off in some software update) using the following steps:

  • get TIConnectCE if you haven't already
  • get arTIfiCE (just follow the instructions on the page)
  • optional but highly recommended: get a more advanced shell like Cesium
